Code puzzles to try to save Christmas!

## Setup
1. Install Ruby version
    `rvm install 3.0.3`
2. Install Bundler
    `gem install bundler`
3. Install dependencies
    `bundle`

## Run
Just run the main file for the day in question:
`ruby ./day_x/main.rb`

## Test
Run the spec file for the day:
`rspec ./day_x/spec/spec_name.rb`
